# 90-second demo script (TASKS §16.1)

1. **Teacher (15s)** Open app → Home → **Teacher mode** → PIN `4242` → **Create quest** → save → note pairing code dialog.
2. **Learner hub (10s)** Back to hub → sample **Food** quest card → **Start**.
3. **Play (40s)** Complete one cloze correctly. On a later task, submit a **wrong answer twice** → read rule hint → **AI multilingual hint sheet** → Continue → finish or skip to session end.
4. **Summary (15s)** Session summary → stars + stats → **Back to hub**.
5. **Teacher insight (10s)** Teacher mode → **Insight cards** (after at least one session with weak-skill signal) or **Class summary**.

Rehearse on a tablet width; enable **TTS** in Settings for the listen-along track (TASKS §12 acceptance).
